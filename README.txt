(Partially copied from http://facepunch.com/showthread.php?t=980687)

- Uninstall or disable Metamod by moving metamod.vdf from garrysmod/addons/ to garrysmod/addons/metamod/.
- Install MySQLOO. Get it here: http://facepunch.com/showthread.php?t=1220537
- Place libmySQL.dll orangebox/ and gmsv_mysqloo.dll in orangebox/garrysmod/lua/includes/modules/
- Set up your Sourcebans settings in lua/autorun/server/sourcebans_setup.lua (open in a text editor)
